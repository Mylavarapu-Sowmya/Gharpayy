# GharPayy Lead Connect

## Overview

GharPayy Lead Connect is a modern web application designed to manage and connect potential leads efficiently.
The platform provides a structured interface for handling lead information, improving communication, and streamlining the lead management workflow.

This project is built with a modern frontend stack to ensure fast performance, scalability, and a responsive user interface.

---

## Technologies Used

* **React** – For building interactive user interfaces
* **TypeScript** – For type-safe JavaScript development
* **Vite** – Fast development server and build tool
* **Tailwind CSS** – Utility-first CSS framework for responsive styling
* **shadcn/ui** – Reusable and customizable UI components

---

## Features

* Lead management interface
* Clean and responsive UI design
* Fast loading with Vite development server
* Modular component structure
* Scalable frontend architecture

## Installation and Setup

Follow these steps to run the project locally.

### 1. Clone the Repository

```
git clone https://github.com/Mylavarapu-Sowmya/gharpayy-lead-connect-main.git

```
### 2. Navigate to the Project Folder
```
cd gharpayy-lead-connect-main
```
### 3. Install Dependencies
```
npm install
```

### 4. Start the Development Server
```
npm run dev
```
